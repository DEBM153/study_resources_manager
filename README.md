# StudyVault 学习资料管理器

StudyVault 是一个使用 Python 编写的命令行学习资料管理工具。当前第一版主要用于记录、查询、整理和统计个人学习资料，适合作为 Python 基础语法、文件读写、JSON 数据保存和 Git 项目管理的练习项目。

程序的数据保存在 `data/resources.json` 中。用户通过终端菜单进行操作，关闭程序后，已经保存的学习资料仍会保留。

## 当前功能

程序启动后会显示菜单，当前支持以下功能：

```text
1. 添加学习资料
2. 搜索所有资料
3. 查看所有资料
4. 按关键词搜索资料
5. 自动整理文件夹
6. 查看学习资料统计
7. 删除资料记录
8. 退出程序
```

### 添加学习资料

用户可以录入一条学习资料记录，包括：

- 资料名称
- 所属科目
- 文件路径
- 标签

标签支持多个，输入时使用英文逗号分隔。

### 搜索资料

程序支持按照指定字段搜索资料：

- `name`：按资料名称搜索
- `subject`：按科目搜索
- `path`：按路径搜索
- `tag`：按标签搜索

只要输入内容包含在对应字段中，程序就会输出匹配到的资料记录。

### 查看所有资料

程序会读取 `data/resources.json` 中保存的全部资料，并按编号逐条显示。如果当前没有资料，会提示“目前没有资料存入”。

### 按标签搜索资料

用户可以输入标签关键词，程序会在所有资料的标签列表中查找对应资料，并输出匹配结果。

### 自动整理资料记录

当前版本的“自动整理”主要整理数据记录本身，不会移动电脑中的真实文件。它会：

- 去掉资料名称和科目前后的多余空格
- 去掉标签中的多余空格
- 删除重复标签
- 对标签进行排序
- 按科目、标签和资料名称对资料记录排序

整理完成后，结果会重新保存到 `data/resources.json`。

### 统计资料

程序会统计：

- 每个科目下的资料数量
- 每个标签出现的次数
- 科目总数
- 标签总数

### 删除资料

用户可以输入资料名称删除记录。删除前程序会展示对应资料，并要求输入 `YES` 确认删除；输入 `NO` 则取消删除。

## 项目结构

```text
Study Srouce manage/
├── main.py
├── README.md
├── .gitignore
└── data/
    ├── functions_menus.txt
    └── resources.json
```

文件说明：

- `main.py`：程序入口，包含菜单显示、资料增删查改、统计和整理功能
- `data/functions_menus.txt`：菜单文本配置文件
- `data/resources.json`：学习资料数据文件
- `.gitignore`：Git 忽略配置，排除缓存、虚拟环境和 IDE 临时文件

## 运行方法

确认电脑已经安装 Python 后，在项目根目录运行：

```bash
python main.py
```

如果系统中 `python` 命令不可用，可以尝试：

```bash
py main.py
```

## 数据格式

每条学习资料在 JSON 中保存为一个对象，当前字段如下：

```json
{
    "name": "python学习指南",
    "subject": "python",
    "path": "data\\python\\py.txt",
    "tag": [
        "python",
        "学习"
    ]
}
```

字段含义：

- `name`：资料名称
- `subject`：所属科目
- `path`：资料路径
- `tag`：资料标签列表

## 当前版本说明

当前第一版已经完成了学习资料管理器的核心命令行流程，包括资料添加、查看、搜索、删除、整理、统计和本地 JSON 持久化保存。

目前程序仍是基础版本，适合个人本地使用。后续可以继续改进：

- 增加输入校验，减少空输入或错误输入导致的问题

- 给每条资料增加唯一编号，删除时可以按编号删除
- 增加创建时间、修改时间等字段
- 将功能拆分到多个 Python 文件中，提升代码可维护性
- 后续可以考虑增加图形界面或网页界面
